Describe your plan and reasoning behind it to answer this question. Which other variables would likely also change?

Answer: Prefiction: if a selection pressure would cause the neocortex fraction to increase from currently 0.755 to 0.770 in the future, the milk enerfy density in human would become higher.

My plan: 1、Fit a linear model between N and E. 2. Obtain the coefficient estimates from linear model, using the coeficient estimates, predict change in E when given the change in N. 3. Calculate the predict value of E using the new value of N (0.770). 4. Compare the value in different value of N.

Reasoning: 1. The linear model provides an estimate of the relationship between two variables based on the observed data. 2. by using the coefficients estimates from the model, we can predict the change in E associated with a given change in N.

The possible changed variables are: brain size, higher metabolic rate and longer lifespan or lower body mass.
